# Michael Barany's dotfiles

## Dependencies
- [Homebrew](http://brew.sh/)
- [GPG](https://gpgtools.org/)

## Installation
```bash
git clone git@github.com:mbarany/dotfiles.git ~/.dotfiles
~/.dotfiles/install
```
